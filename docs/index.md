# Welcome to coreason-publisher

**coreason-publisher** is the Regulatory Gatekeeper and Artifact Packager for the CoReason platform. It is responsible for orchestrating the release process of AI agents in GxP environments, ensuring compliance, reproducibility, and governance.

## Documentation Overview

*   **[Architecture](architecture.md):** Learn about the core philosophy, functional components, and the "Package-Propose-Approve-Merge" loop.
*   **[Usage](usage.md):** Step-by-step guide on how to use the CLI for proposing and finalizing releases.
*   **[Requirements](requirements.md):** The Product Requirements Document (PRD) detailing the specifications and user stories.

## Quick Start

Install the package via Poetry:

```bash
poetry install
```

Check the [Usage](usage.md) section for CLI commands.
